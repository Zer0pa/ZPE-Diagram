# FAQ

## Does this support arbitrary SVG?

No. The bounded proof packet is for line-based diagram inputs inside the documented scope.

## What style information is preserved?

Only the frozen 8-color palette plus quantized stroke width. Draw order is preserved as explicit state.

## What is out of scope?

Fill, dashed input on the encode path, taper, pressure, and out-of-palette colors.

## Is this a general drawing or illustration tool?

No. This repo is a bounded encoding product, not a general-purpose SVG editor.
