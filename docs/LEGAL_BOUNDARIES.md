# Legal Boundaries

This document states the public product boundary for `zpe-diagram`.

This repo is distributed under `Zer0pa Source-Available License v7.0`. See `LICENSE` at the repo root.

This repo exposes one bounded adopter surface:

- structural diagram encoding
- bounded style encoding for the frozen palette plus quantized width
- explicit draw-order preservation

This repo does not claim broader SVG support, general illustration semantics, fill support, dashed encode-path support, taper, or out-of-palette color identity.
