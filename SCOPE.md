# Scope

`zpe-diagram` is a bounded adopter for structural-with-style diagram encoding.

## In Scope

- structural line geometry for controlled line-based SVG inputs
- bounded style preservation for the frozen 8-color palette
- quantized stroke width preservation
- draw order as explicit state
- standalone validation and focused tests for this bounded surface

## Out of Scope

- fills
- dashed input on the encode path
- taper or pressure variation
- out-of-palette color identity
- arbitrary SVG coverage
- general drawing or illustration semantics

## Change Rule

Any expansion of the public claim needs new committed proof artifacts before it belongs in this repo.
